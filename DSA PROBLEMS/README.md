# Data Structures and Algorithms (DSA) Problems

This repository contains a curated collection of Data Structures and Algorithms problems, categorized by topic. Each solution includes a detailed **Editorial** with intuition, approach, and complexity analysis.

## 📂 Repository Structure

The problems are organized into the following categories:

- **[ARRAYS](./ARRAYS/)**: Binary Search, Two Pointers, Prefix Sum, Sliding Window, and Kadane's Algorithm.
- **[LINKED LISTS](./LINKED%20LISTS/)**: Fast & Slow Pointers, Merging, Deletion, and Reversal.
- **[STACKS](./STACKS/)**: Balanced Parentheses, Tower of Hanoi, and Stack Reversal.
- **[STRINGS](./STRINGS/)**: String manipulation, Edit distance, and Palindrome checks.
- **[Greedy Algorithms](./greedy%20algo/)**: Optimization problems like Jump Game and Container With Most Water.
- **[Queues](./QUEUES/)**: Implementation and applications of Queue data structures.
- **[Recursion](./RECURSION/)**: Fundamental recursive problems and backtracking.
- **[Bit Manipulation](./BIT%20MANUPULATION/)**: Efficient bit-level operations.

## 🚀 Mini Projects

Beyond standard DSA problems, this folder also includes mini-projects implemented in C++:

1. **[Number Guessing Game](./project2.cpp)**: A game that utilizes logical ranges (similar to Binary Search) to guess a secret number.
2. **[Word Scramble Game](./project3%20wordscramblegame.cpp)**: An interactive country-guessing game using string shuffling techniques.

## 📝 Problem Documentation

Each `.cpp` file is structured as follows:
- **Problem Statement**: A clear description of the task.
- **Editorial**: 
    - **Intuition**: The "why" behind the solution.
    - **Approach**: Step-by-step logic.
    - **Complexity**: Time and Space analysis.
- **Solution**: Clean and optimized C++ code (often including both Brute Force and Optimal approaches).

## 🛠️ Usage

To run any of the C++ solutions, ensure you have a C++ compiler (like `g++`) installed.

```bash
g++ "FileName.cpp" -o output
./output
```
